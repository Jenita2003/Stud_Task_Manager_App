# Student Task Manager App

## Project Description
This is a simple Android application built using Java in Android Studio that helps users manage daily tasks efficiently.

Users can:
- Login
- add, edit, search, delete, and mark tasks as completed.

-----------------------------------------------------------------------------------------------------------------------------------------

## Features

- Login screen with basic validation
- Home screen with task list
- Add new tasks (title, description, due date)
- Search their tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as Completed / Pending
- RecyclerView for displaying tasks
- Dark mode UI 

-----------------------------------------------------------------------------------------------------------------------------------------

## Technologies used

- Java
- Android Studio
- Database: Data was maintained by ArrayList during runtime (as of the requirement)
- Version Control: Git
- Repository: Github

-----------------------------------------------------------------------------------------------------------------------------------------

## Steps to Run the Project

1. Open the project in Android Studio

2. Connect your mobile phone to the laptop using a USB cable

3. Turn on Developer Mode in your phone   and enable USB Debugging
   (Go to About Phone → tap Build Number 7 times)

4. Allow the permission popup that comes on the phone and select your mobile device in Android Studio

5. Click on the Run button

6. The app will install and run on your mobile phone


## --- Note ---
The app was tested by running directly on a mobile device using USB connection from Android Studio instead of emulator
